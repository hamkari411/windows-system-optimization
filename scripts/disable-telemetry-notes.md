# Telemetry Review Notes

This document outlines common steps used to review and manage telemetry settings in Windows environments.

## Service Review

Common services evaluated:

- Connected User Experiences and Telemetry
- Diagnostic Policy Service
- Windows Error Reporting Service

## Startup Impact

- Review startup applications in Task Manager
- Disable non-essential programs

## Privacy Settings

- Disable advertising ID
- Turn off background app permissions
- Limit diagnostic data level

Note:
Changes should always be documented and tested before large-scale deployment.
